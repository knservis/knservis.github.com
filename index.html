Computing as profession

Currently I am employed as a Software Developer at HIE. My duties involve maintaining and improving a number of web applications that the company is deploying. The deployment platform is Linux and the main language used is Ruby. In my job I try to use the best practices that I know of such as Test and Behaviour Driven Development without becoming religious about them. I try to avoid reinventing the wheel and try to dissuade others form doing so, as I think that in general one is best off using code that others have written, tested and are using. If there are issues with the library you are using, submit a patch!

My main interests are web information systems, machine learning and artificial intelligence. Most of my professional experience is in web development and most of the courses I have studied had something to do with machine learning and related areas such as data mining and AI in general. Some pages or web applications that I have helped create or have created are: http://www.mirrabooka.com, http://www.zentro.com.au, https://myapnic.net (users only).
